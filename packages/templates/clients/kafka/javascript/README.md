## JavaScript Kafka Client

You can test this template:
1. Clone the project and run `npm install`
2. Navigate to `packages/templates/clients/Kafka/test/integration-test` and run the tests with `npm run test` to generate Kafka clients
3. Navigate to `packages/templates/clients/Kafka/javascript`
4. Install with `npm install` and run test with `npm run test`
5. Start example script that uses a client library generated by the test: `node example.js`
