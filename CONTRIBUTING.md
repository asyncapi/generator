# Contributing to AsyncAPI

The full contributing guide is now part of the official documentation, published at **[asyncapi.com/docs/tools/generator/contributing](https://www.asyncapi.com/docs/tools/generator/contributing)**. The source lives in [`apps/generator/docs/contributing.md`](apps/generator/docs/contributing.md). This file is a short index.

The guide covers:

- Contribution recognition and guidelines for new contributors
- The contribution flow, issues, and pull requests
- [Conventional Commits](https://www.asyncapi.com/docs/tools/generator/contributing#conventional-commits) for commit and PR titles
- AI-assisted contributions and the AI tools the project runs on your PRs and issues
- Maintainer roles: Triager, Committer, and Champion

See also the [AI Usage Policy](https://www.asyncapi.com/docs/tools/generator/ai-policy) and the [Code of Conduct](CODE_OF_CONDUCT.md).
